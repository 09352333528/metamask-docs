Class: KeyringSnapControllerClient

A KeyringClient that allows the communication with a snap through the SnapController.

Hierarchy​

  • KeyringClient

    ↳ KeyringSnapControllerClient

Constructors​

constructor​

• new KeyringSnapControllerClient(args)

Create a new instance of KeyringSnapControllerClient.

The handlerType argument has a hard-coded default string value instead of a HandlerType value to prevent the @metamask/snaps-utils module from being required at runtime.

Parameters​

NameTypeDescription
argsObjectConstructor arguments.
args.controllerSnapControllerThe SnapController instance to use.
args.handler?HandlerTypeThe handler type (default: 'onRpcRequest').
args.origin?stringThe sender's origin (default: 'metamask').
args.snapId?stringThe ID of the snap to use (default: 'undefined').
