buildHandlersChain​

▸ buildHandlersChain(...handlers): OnRpcRequestHandler

Build a chain of handlers for a JSON-RPC request.

If a handler throws a MethodNotSupportedError, the next handler in the chain is called. If all handlers throw a MethodNotSupportedError, the error is re- thrown.

Any other error thrown by a handler is re-thrown.
